# Hướng dẫn dùng LearnForge

Hướng dẫn này dành cho người dùng cuối — từ đăng ký đến hoàn thành khóa học đầu tiên.

---

## Đăng ký tài khoản

1. Vào trang chủ → bấm **"Bắt đầu học"**
2. Chọn:
   - **Đăng ký bằng email** — điền email + mật khẩu (min 8 ký tự)
   - **Đăng nhập với Google** — nếu admin đã bật tính năng này
   - **Đăng nhập với GitHub**
3. Sau khi đăng ký: đi qua **onboarding** (3 bước ngắn)

---

## Onboarding (lần đầu đăng nhập)

Onboarding chỉ hiện **1 lần** khi bạn đăng ký mới:

1. **Chọn chủ đề** — bạn muốn học gì trước? (bạn có thể đổi sau)
2. **Đặt mục tiêu** — học 5 / 10 / 15 phút mỗi ngày
3. **Chọn avatar** — mascot đồng hành trong hành trình học

---

## Upload tài liệu và tạo khóa học

LearnForge biến **bất kỳ tài liệu nào** thành khóa học tương tác.

### Loại tài liệu hỗ trợ

| Loại | Ví dụ |
|---|---|
| PDF | Slide bài giảng, sách giáo khoa, tài liệu kỹ thuật |
| DOCX | File Word, giáo trình |
| TXT / Markdown | Notes, cheatsheet |
| URL | Bài viết blog, tài liệu online |
| YouTube | Paste link video → lấy transcript tự động |

### Cách upload

1. Vào **Upload** (menu trái)
2. Kéo thả file vào vùng upload, hoặc bấm để chọn file
3. Đặt tên khóa học
4. Bấm **"Tạo khóa học"**
5. Đợi hệ thống xử lý (thường 1–3 phút tùy kích thước tài liệu)

**Trạng thái xử lý:**
- `Processing` — đang phân tích tài liệu
- `Generating` — AI đang tạo chương trình học
- `Ready` — khóa học sẵn sàng

> **Free tier:** tối đa 3 tài liệu. Muốn nhiều hơn: liên hệ admin để nâng cấp Pro.

## Học bài

### Learning Map

Sau khi khóa học tạo xong, bạn thấy **Learning Map** — bản đồ các bài học dạng nút.

- **Nút xanh** — bài học đã mở, chưa học
- **Nút vàng** — đang học
- **Nút xám** — chưa mở (cần hoàn thành bài trước)
- **Nút có ngôi sao** — checkpoint lesson (khó hơn, thưởng nhiều XP hơn)

Bấm vào nút để xem preview bài học, rồi bấm **"Bắt đầu"** để học.

### Màn hình bài tập

- **Thanh tiến độ** ở trên cùng — bao nhiêu bài tập còn lại
- **Hiển thị Hearts** góc trên phải — bạn còn bao nhiêu tim
- Mỗi bài tập hiện một lần, trả lời xong mới sang bài tiếp

### 6 loại bài tập

| Loại | Cách làm |
|---|---|
| Trắc nghiệm | Chọn 1 trong 4 đáp án |
| Điền vào chỗ trống | Gõ từ hoặc cụm từ còn thiếu |
| Ghép đôi | Kéo thả để ghép cột trái với cột phải |
| Sắp xếp thứ tự | Kéo thả để sắp xếp các bước theo đúng thứ tự |
| Điền code | Hoàn thành đoạn code còn thiếu |
| Đúng / Sai | Chọn True hoặc False, có giải thích |

### Feedback sau mỗi câu

- **Đúng** → thanh xanh, +điểm, qua bài tiếp
- **Sai** → thanh đỏ, -1 heart, hiện giải thích đúng

## Hệ thống Gamification

### Hearts (Tim)

- Bạn có **5 hearts** mặc định
- Mỗi câu trả lời sai → -1 heart
- Hearts tự phục hồi: **1 heart / 30 phút**
- Hết hearts → không học được, phải đợi hoặc dùng gems

**Lấy lại hearts:**
- Đợi tự phục hồi
- Dùng 150 gems → phục hồi đầy đủ ngay
- Tài khoản Pro → unlimited hearts

### Streak (Chuỗi ngày học liên tục)

- Học ít nhất 1 bài/ngày → streak +1
- Bỏ 1 ngày → streak về 0
- **Streak Freeze** (100 gems) → bảo vệ streak qua 1 ngày bỏ học

**Milestone thưởng:**
- 7 ngày liên tục → +30 gems
- 30 ngày → +100 gems
- 100 ngày → huy hiệu đặc biệt

### XP và Gems

**Kiếm XP:**
| Hoàn thành | XP |
|---|---|
| Bài học thường | +10 XP |
| Bài học không sai lần nào | +15 XP |
| Checkpoint lesson | +25 XP |

**Kiếm Gems:**
| Cách | Gems |
|---|---|
| Bài học không sai lần nào | +5 gems |
| Checkpoint lesson | +15 gems |
| Daily quest | +10–30 gems |
| Streak 7 ngày | +30 gems |

### Cửa hàng (Shop)

Dùng gems để mua:
| Item | Giá |
|---|---|
| Streak Freeze | 100 gems |
| Phục hồi Hearts | 150 gems |
| Weekend Shield | 200 gems |
| Cosmetic themes | 500 gems |

### Daily Quests

Mỗi ngày có 3 quest ngắn, reset lúc nửa đêm:
- Ví dụ: "Hoàn thành 3 bài học" → +20 gems
- Xem quests: bấm icon cờ ở sidebar

## AI Companion

AI Companion là chatbot chỉ trả lời dựa trên tài liệu **của bạn**.

**Dùng để làm gì:**
- Hỏi về nội dung tài liệu bạn đã upload
- Giải thích khái niệm chưa hiểu
- Ôn tập trước khi làm bài tập

**Vào Companion:** menu trái → biểu tượng chat.

> Companion **không trả lời** câu hỏi ngoài phạm vi tài liệu của bạn. Đây là tính năng bảo mật dữ liệu, không phải lỗi.

## Cài đặt tài khoản

Bấm avatar góc dưới trái → **"Settings"**

- **Profile** — đổi tên, avatar
- **Goals** — đổi mục tiêu học hàng ngày
- **API Keys** — thêm key cá nhân (xem [BYOK-GUIDE.md](BYOK-GUIDE.md))
- **Model Config** — chọn AI model cho từng tác vụ
